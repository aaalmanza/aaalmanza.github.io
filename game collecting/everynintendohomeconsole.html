This is pretty self-explanatory. I own every generation of Nintendo home consoles, from the Nintendo Entertainment System which released in 1985, to the Nintendo Switch 2 which released just this year. It has been a long journey to get each of these systems, starting around 2019 when I scored a Nintendo Entertainment System in the box (albeit in very rough condition) at Goodwill. From top left to bottom right, the systems in order are the Nintendo Entertainment System, Super Nintendo Entertainment System, Nintendo 64, GameCube, Wii, Wii U, Switch, and Switch 2.
